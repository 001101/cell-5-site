legacy code uplift

We embrace working with existing code. Examples of our uplifts include:

  • Migration from on-premises java monolith to AWS based clojure microservices

  • Reverse engineering previous Wordpress sites and revitalizing them

  • Migration from Cobol mainframe to Java Spring services

  • Migration from proprietary version control to git

  • Migration from on-prem tooling to cloud tooling

  • Migration from legacy Jquery / KnockoutJs to VueJs / ReactJs

We are suprised how reluctant some developers are to re-use, and re-vitalize something existing.

Our principles and approaches when working with legacy are:

  • The previous developers did the best they could at the time they did it.

  • We break things up in small steps, iteratively, keeping things working as we go.

  • We do things safely, guided by tests and automation.

If you are stuck with something legacy and need a hand then we are here to help!
